# Etch-a-Sketch

## Overview
A simple browser-based drawing app inspired by the classic Etch-a-Sketch toy. Move your mouse over the grid to “draw” by changing the color of the squares.

## Features
- 16×16 grid generated on page load
- Hover over squares to fill them with color
- Change grid size (up to 100×100)
- Squares automatically resize to fit the container
- Clear button to reset the drawing area

## Live Demo
Link: https://brodr2932.github.io/etch-a-sketch/

## How to Use
1. Open the webpage in your browser
2. Move your mouse over the squares to draw
3. Click “Change Grid Size” to create a new grid
4. Click “Clear Grid” to erase your drawing
